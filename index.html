Mise en place

Configuration rapide

  1. Créer un projet Firebase et activer Authentication / Google ainsi que Firestore.
  2. Vérifier que config.js pointe bien vers le bon projet Firebase.
  3. Activer GitHub Pages sur la branche GitHub Actions : le workflow publiera l’application.
