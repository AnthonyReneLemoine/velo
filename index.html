Mise en place

Configuration rapide

  1. Créer un projet Firebase et activer Authentication / Anonyme et Firestore.
  2. Remplir config.js avec la configuration web Firebase et le chemin du document.
  3. Activer GitHub Pages sur la branche GitHub Actions : le workflow publiera l’application.
